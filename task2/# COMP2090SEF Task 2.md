# COMP2090SEF Task 2
Self-study on a New Data Structure and Algorithm

## Topics
Data Structure: Heap  
Algorithm: Cocktail Sort

## Description
This project studies a new data structure and algorithm that are not covered in the course curriculum.

The data structure selected is **Heap**, which is widely used in priority queues and efficient data management.

The algorithm selected is **Cocktail Sort**, a variation of Bubble Sort that sorts the list in both directions.

## Algorithm Complexity

Cocktail Sort

Best Case: O(n)  
Average Case: O(n²)  
Worst Case: O(n²)
